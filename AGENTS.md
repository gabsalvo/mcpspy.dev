# About this repo

This is the mcpspy.dev website: a fully static landing page + docs for the open-source mcp-spy CLI. There is no backend, no auth, no database — keep it that way. The design language (paper/ink palette, JetBrains Mono, ASCII art, sunset accents) is shared with gabsalvo.com and domoroshi.tech; design tokens live in `app/globals.css`.
